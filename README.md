# 📸 Upload de Projetos

Uma interface web simples que permite criar e montar visualmente uma **publicação de uma rede social de projetos** — onde você pode inserir informações e estruturar um post de projeto diretamente no navegador.

Este projeto foi feito com **HTML, CSS e JavaScript puro**, ideal para entender como construir interfaces interativas leves sem frameworks.

## 📌 Sobre o Projeto

Esta aplicação apresenta uma interface para montar um post de projeto — um modelo de publicação que poderia ser usado em uma rede social ou portfólio. O foco é a interação com o usuário através de formulários e visualização imediata do conteúdo que está sendo criado.

### 🧩 O que ele faz

✔ Permite ao usuário inserir conteúdo para um post  
✔ Atualiza dinamicamente elementos na interface  
✔ Exibe o resultado em uma área de visualização  
✔ Tudo em tempo real com **Javascript manipulando o DOM** 

## 📂 Estrutura dos Arquivos

| Arquivo | Descrição |
|---------|-----------|
| `index.html` | Estrutura da página e formulários |
| `styles.css` | Estilos visuais da interface |
| `scripts.js` | Lógica que atualiza a interface com os dados do usuário |
| `img/` | (possível pasta de imagens) |

## 🛠 Tecnologias Utilizadas

- **HTML5** – marcação da interface  
- **CSS3** – estilização visual  
- **JavaScript (DOM)** – lógica interativa  

Não há dependências externas — tudo roda no navegador!
